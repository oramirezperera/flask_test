Hello and welcome!

You can Use this app as it follows

  • For a random Dad Joke
  • For a random Chuck Norris Joke
  • For number
  • Numbers

Random Dad Joke returns a Dad joke from icanhazdadjoke.com

Random Chuck Norris joke returns a Joke from api.chucknorris.io

In number you pass in the URL an integer and returns that integer + 1. If you don't pass any number the example code runs with 1 + 1.

In number you pass in the URL a list of numbers separated by ',' and returns the least common multiple of them. If you don't pass any parameter in the url the example code runs with the list 2,3.
